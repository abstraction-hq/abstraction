# OpenPass
OpenPass is open source wallet built on top of account abstraction.

## Features
- Account Abstraction
- Passkey Authentication
- Password recovery
- Seamless defi experience

## Components
### Web application
Web application to interact with OpenPass wallet.

### Indexer
Indexer to index OpenPass wallet transactions and positions.

### API
API serve information about OpenPass wallet transactions and positions.
